== Descripción ==
Plugin desarrollado por [iCornio](https://icornio.com) 
Desarrollador : Juan Giovanni John ([GitHub](https://github.com/juangiovannijohn) )
Cliente: Siscard SA

Plugin que automatiza la carga en la base de datos de técnicos autorizados por Siscard SA.
Utilizando un archivo .CSV se cargan automaticamente toda la información sobre los tecnicos, Cuenta ademas con una funcionalidad para la asignacion automatica de imagenes de perfil para cada tecnico.

== Instalación ==

Ingresar al repositorio publico del autor [Link](https://github.com/juangiovannijohn/plugin_import_csv)

1. **Click** Code > Download ZIP
2. **Pre-configuraciones del sitio WP** Leer la Documentación punto 2.
3. **Dashboard WP** Plugins > Agregar Nuevo > Subir plugin > Cargar .zip descargado y Activar
4. **Crear archivo .csv** En el plugin hay un ejemplo del archivo Excel que debe transformarse a CSV para luego subirlo.
